The accounting process

When

Use the first and second week to submit reimbursements, do the bookkeeping on the second week’s Thursday afternoon, next time: 2023/08/10

The steps to make monthly record

Excel processing

  1. Dump the record from Mint to nyuzyt/Personal file/Transactions cvs file
    • post credit karma’s acquisition of mint, use creditkarma_export_transactions to download the transactions. Read the instruction.
    • remove the previous record files first
  2. Insert a line in overall sheet prior to the previous record, then insert the copied cells
  3. Copy the cvs tables content, without the header
  4. Create a new sheet for this month alone
  5. Copy from one previous month’s sheet(all the way to comments), paste the column width
  6. Copy the records to this month’s sheet
  7. Remove the Credit Card Payments
  8. Remove the Loan accounts
  9. Filter the records by account first
  10. Check each account, modify the category and mark the transactions as verified
  11. Copy the reverse ~ Balance columns from one previous month’s table
  12. List single item takes more than $10,000

Web update

  1. Generate the new hexo post with hexo new post "Bookkeeping - XXXX/XX"

  2. Copy all content from the post for previous month, git add now(so we know what changed)

  3. Commit the change with title Add bookkeeping for YYYY/MM

  4. Go to the Table to Markdown website to transform excel tables to markdown tables

  5. Remove the zeroes from sum column

  6. Open the same file to the right window, make it narrow

  7. Add new value to and remove old value from the multi layer pie chart for spending and income

  8. Ask chatgpt to generate the detailed category data

  9. Add new value to and remove old value from the trend
